1. About Fusion 3D Template

Fusion 3D is a premium e-commerce template featuring stunning 3D text effects, smooth animations, and a modern minimalist design. Perfect for fashion brands, design studios, art galleries, and premium product showcases.

This template is compatible with Ecwid by Lightspeed.

Key Features

  • Interactive 3D text — Hero section with customizable 3D fonts and lighting
  • Dynamic lighting — Three light sources for depth and dimension
  • Smooth animations — Float and wave effects with speed control
  • Interactive map — Show multiple store locations with Google Maps
  • Product gallery — Beautiful grid layouts with hover effects
  • Collections — 3D flip-card hover effects
  • Reviews carousel — Customer testimonials with auto-scroll
  • Fully responsive — Works on all devices

Lighting

The template includes three light sources (Light 1, Light 2, Light 3) that add depth and dimension to your 3D text. Toggle them on or off to achieve different visual effects. More lights = more depth and highlights on your text.

3D Text Settings

Setting Type Description
3D Title Text The text that will be rendered in 3D
3D Title Size Range Font size (12-60px). Reduce if text doesn't fit on screen.
Font Select Choose from supported 3D fonts (see list above)
Light 1, Light 2, Light 3 Toggle Turn lights on/off to add depth to 3D text
Metal Material Toggle Makes text look metallic and reflective
3D Animation Toggle Enable/disable animation effect
Animation Type Select Float — gentle bobbing
Wave — wave-like rotation
Animation Speed Select Slow (0.5x), Normal (1x), Fast (2x)
Text Not Fitting on Screen?

If your 3D text is too large and doesn't fit properly, simply reduce the 3D Title Size in Design settings until it looks right.

Mobile Size

On mobile devices, the 3D text size is automatically set to 14 and cannot be changed. This ensures optimal performance and readability on smaller screens, as 3D rendering is resource-intensive on mobile devices.

3. Product Gallery

The product gallery showcases your products with beautiful hover effects. You can add up to 8 images and link each to a product page.

Manual Product Setup

The gallery is not connected to your Ecwid catalog directly. This gives you full control over which products to highlight and lets you use any images — such as lifestyle photos or custom photoshoots — not just standard catalog images.

How to Get Product Links

  1. Go to your Ecwid admin panel → Catalog → Products
  2. Click on the product you want to link
  3. Copy the URL from your browser's address bar
  4. Paste it into the Image Link field in the gallery settings

You can use either the full URL or relative path:

https://your-store.company.site/products/product-name

or just:

/products/product-name
Quick Tip

Both formats work! Use relative paths (/products/...) for links within your store, or full URLs for external websites.

4. Collections

Display your product collections with a 3D flip-card hover effect. You can add up to 8 collection cards.

How to Get Collection Links

  1. Go to your Ecwid admin panel → Catalog → Categories
  2. Click on the category/collection
  3. Copy the URL from your browser's address bar
  4. Paste it into the Collection URL field

You can use either the full URL or relative path:

https://your-store.company.site/products/collection-name

or just:

7. Location & Map

Display an interactive map with your store locations. Visitors can see details and get directions.

How to Get Coordinates from Google Maps

Each location needs GPS coordinates. Here's how to get them:

  1. Open Google Maps in your browser
  2. Search for your store address or navigate to the exact location
  3. Right-click directly on the location on the map
  4. Click on the coordinates that appear at the top of the menu — they will be copied automatically
  5. Paste the coordinates into the Coordinates field in the template settings
Open Google Maps

Example: Getting coordinates

Right-click on the map → Click the coordinates (e.g., 40.7484, -73.9857)

Coordinates Format

Coordinates should be in decimal format: latitude, longitude

Examples:
• New York: 40.7128, -74.0060
• London: 51.5074, -0.1278
• Paris: 48.8566, 2.3522
• Tokyo: 35.6762, 139.6503
• Sydney: -33.8688, 151.2093

Location Settings

Setting Type Description
Default Map Location Text Initial center point of the map (coordinates)
Map Zoom Select Far — Wide area view
Normal — Balanced view
Close — Street level
Very Close — Building detail
Location Name Text Store name (e.g., "Tokyo Flagship")
Address Text Full street address
Phone Text Contact phone number
Coordinates Text GPS coordinates (latitude, longitude)
Map Not Showing?

If the map doesn't display:

1. Make sure coordinates are in correct format: 40.7128, -74.0060
2. Don't include degree symbols or letters (°, N, S, E, W)
3. Use a decimal point, not a comma, for decimal places
4. Latitude comes first, then longitude

9. Troubleshooting

Solutions to common issues with the Fusion 3D template.

3D text not displaying or looks broken

Cause: You're using an unsupported font.

Solution: Only use fonts from the supported fonts list. Best options: Smooch, Luckiest Guy, Grandstander, Anton.

3D text is too large / doesn't fit on screen

Solution: Reduce the 3D Title Size in Design settings until it fits properly.

Note: On mobile devices, the size is fixed at 14 for performance reasons and cannot be changed.

3D text looks flat / no depth

Solution: Enable more lights (Light 1, Light 2, Light 3) to add depth. You can also try enabling "Metal Material" for a more visible 3D effect.

Map not showing / wrong location

Solution: Check your coordinates format:

  • Use decimal format: 40.7128, -74.0060
  • Latitude first, longitude second
  • No degree symbols (°) or letters (N, S, E, W)
  • Use period (.) for decimals, not comma
Product links not working

Solution: You can use either format:

  • Full URL: https://your-store.company.site/products/product-name
  • Relative path: /products/product-name

Copy the URL from your browser when viewing the product page.

Animation is too slow / too fast

Solution: Change the Animation Speed setting:

  • Slow = 0.5x speed
  • Normal = 1x speed
  • Fast = 2x speed
Images not loading in gallery

Solution: Make sure images are:

  • In JPG, PNG, or WebP format
  • Under 5MB file size
  • Properly uploaded through the image picker
